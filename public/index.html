Disabled buttons

Being able to disable a button might come quite handy. Works only with button elcodeents that have disabled attribute.

Button groups

You can group buttons by using .button-group class. For outlined versions use .outlined class along with the color denotation – .primary or .secondary – if you would like to change the default appearance.

Forms

Inputs and textareas

All elcodeents have predefined :focus state and placeholders.

Validity

You can keep inputs in .input-wrapper (recommended). And add validity classes of valid and invalid which will change the styles of labels, inputs and icons (if present) accordingly.

If you are willing to use icons make sure the wrapper elcodeent also has a class .has-icon.Then add appropriate icons with this markup: i.ss-icon.ss-standard.ss-nameoftheicon

Other form elcodeents

All basic elcodeents have been covered: file inputs, checkboxes, radio buttons and selects. Despite checkboxes and radios, where you should add a.disabled class to the label you can use the:disabled attribute.

Overlay

If you need to cover the content of the page to display the overlay (probably most cases) – you will need an .overlay div appended to body of the page. It will create an scodei-transparent black overlay.
