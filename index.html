Typography

Font families used in majority of &yet's apps are Gotham and Helvetica Neue. When using Gotham be aware of the fact that it's web font served through HFJ Cloud Typography.

To use Gotham update your body font family accordingly and set up a new development project in Typography.com dashboard.

Headings

Headings run from 1—6. Base font size is the equivalent of 16px with em as a primary unit.

Buttons

To create a button add button class to <a> or <button>

Button 
<a href="" class="button">Button</a>
<button class="button">Button</button>

Disabled buttons

To disable buttons add button-disabled class or for <button> elements feel free to use the disabled attribute.

Buttons sizes

Buttons come in three different sizes — large, regular (no class) and small.

Large ButtonButtonSmall Button
<a href="" class="button button-small">Small Button</a>
<a href="" class="button">Button</a>
<a href="" class="button button-large">Large Button</a>

Button groups

Button grouping is available my adding .button-group class. For additional outline include .button-group-outlined class along with the color denotation — .primary or .secondary— if you would like to change the default appearance.

Disabled inputs

To disable a form element add disabled attribute. 

 <input type="text" placeholder="This input is disabled" disabled>

Input validity

Sometimes you need to validate your forms. To have meaningful styling for valid and invalid inputs add input-valid and input-invalid classes respectively. 

<input type="text" class="input-invalid">
<input type="text" class="input-valid">

Modals

Regular Modals

All modals are triggered by having a div with a class .modal. Detailed markup for modal usage can be found in the readme.

Modals with no header

Sometimes you need a simple modal without header and just paragraph with buttons. Use the class .no-head. Paddings will be applied automatically.

Overlay

If you need to cover the content of the page to display the overlay (probably most cases) – you will need an .overlay div appended to body of the page. It will create an semi-transparent black overlay.
