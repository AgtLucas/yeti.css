Typography

Font families used in majority of &yet's apps are Gotham and Helvetica Neue. When using Gotham be aware of the fact that it's web font served through HFJ Cloud Typography.

To use Gotham update your body font family accordingly and set up a new development project in Typography.com dashboard.

Headings

Headings run from 1—6. Base font size is the equivalent of 16px with em as a primary unit.

Buttons

To create a button add button class to <a> or <button>

Button 
<a href="" class="button">Button</a>
<button class="button">Button</button>

Disabled buttons

To disable buttons add button-disabled class or for <button> elements feel free to use the disabled attribute.

Buttons sizes

Buttons come in three different sizes — large, regular (no class) and small.

Large ButtonButtonSmall Button
<a href="" class="button button-small">Small Button</a>
<a href="" class="button">Button</a>
<a href="" class="button button-large">Large Button</a>

Button groups

Button grouping is available my adding .button-group class. For additional outline include .button-group-outlined class along with the color denotation — .primary or .secondary— if you would like to change the default appearance.

Disabled inputs

To disable a form element add disabled attribute.

Input validity

Sometimes you need to validate your forms. To have meaningful styling for valid and invalid inputs add input-valid and input-invalid classes respectively. 

<input type="text" class="input-invalid">
<input type="text" class="input-valid">

Dismissable messages

For messages that can be dismissed add class message-has-icon and appropriate classes for the icon itself — ss-icon.ss-standard.ss-delete (or use a different icon set, if you wish). Do remember to include the title attribute as well.

This is an error that can be dismissed.

This is an alert that can be dismissed.

This is a success message that can be dismissed.

<div class="message message-success message-has-icon">
  <p>This is a success message that can be dismissed</p>
  <a href="" class="ss-icon ss-standard ss-delete", title="Close"></a>
</div>

Full width dismissable messages

Messages spanning for full width of the container should have full-width class.

This is a full width success message that can be dismissed.

<div class="message message-full-width">
  <p>This is a success message that can be dismissed</p>
  <a href="" class="ss-icon ss-standard ss-delete", title="Close"></a>
</div>

Modals

Regular Modals

All modals are divs with class modal and need to be triggered by Javascript. 

<div class="modal">
  <div class="modal-head">
    <h3>Modal Title</h3>
    <a href="" class="modal-close"></a>
  </div>
  <div class="modal-body">
    <p>Modal Content</p>
    <div class="modal-button-wrapper">
      <button>Button</button>
    </div>
  </div>
</div>

Modals with no header

Sometimes you need a simple modal without header and just paragraph with buttons. Use the class modal-no-head and omit the modal-head container.

Overlay

If you need to cover the content of the page to display the overlay (probably most cases) — you will need an overlay div appended to body of the page or has-modal class added to the body element. Both will create an semi-transparent black overlay. 

<div class="overlay"></div>

or

<body class="has-modal"></body>

Avatars

Avatars come in three sizes — large (100×100) , medium (50×50) and small (25×25).

Avatars inside links

They can be wrapped with a link with a avatar-wrapper class. 

<a href="" class="avatar-wrapper">
  <img src="image.jpg" class="avatar avatar-large avatar-rounded" />
</a>

Media

Videos, figures and images have the maximum width of 100% , so they never overflow the container (even on mobile).

Media inside links

You can wrap video and img in a link. 

<a href="" class="media-wrapper">
  <figure>
    <img src="image.jpg" />
  </figure>
</a>

Images with captions

For semantic markup use the figure element with figcaption. 

<figure>
  <img src="image.jpg" />
  <figcaption>Image description</figcaption>
</figure>

Media with an outline

Having an outline around any type of media can be added by using media-outlined class.

<figure class="media-outlined">
  <img src="image.jpg" />
  <figcaption>Image description</figcaption>
</figure>

<figure class="media-outlined">
  <img src="image.jpg" />
</figure>
